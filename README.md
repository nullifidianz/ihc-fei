# Projeto de Interface Humano-Computador

Projeto apresentado ao Centro Universitário [FEI](https://portal.fei.edu.br/), como parte dos requisitos necessários para aprovação na disciplina de Interface Humano-Computador (CC8122) do curso de Ciencia da Computação, orientado pelo Prof. Dr. [Plino Thomaz Aquino Junior](http://lattes.cnpq.br/6186413528999908).  

Este projeto se baseia no Trabalho de Conclusão de Curso (TCC) entitulado **Uso de blockchain e smart contracts para validação de diplomas e certificados acadêmicos** sob orientação do Professor **Leonardo Anjoletto Ferreira** e desenvolvido pelos seguintes alunos:

## Conhecendo o Problema

- Previsto desenvolver Interface? (  x ) Sim     (   ) Não
- Objetivo do trabalho?

Validação de certificados acadêmicos utilizando blockchain.

- Qual o produto final?

Sistema de validação e verificação de certificados e diplomas acadêmicos.

- Quem é o usuário final deste produto?

Universidades, Estudantes.

- O que o usuário recebe de benefício ao usar esse produto?

Ao utilizar o produto o Usuario recebe como benefício um sistema facilitado e decentralizado de verificação de diplomas e certificados.

- Quais as funcionalidades da ferramenta (visão do usuário)?

Como Universidade consigo: Salvar certificados e diplomas e vinculá-los a alunos cadastrados e verificar a validez dos diplomas.

Como Estudante consigo: Visualizar meus diplomas e certificados e verificar a validez dos diplomas.

- Quais tecnologias e ferramentas computacionais que pretendem usar neste projeto (TCC)?

Hyperledger, Blockchain, Solidity, Java, React, DB Relacional.

- Qual é o contexto de uso dessa aplicação? (esse já é um conceito de IHC que professor explicou na sala)

No contexto da Universidade podemos descrever o ambiente como: Calmo, silencioso, bem iluminado (normalmente a secretaria da Instituição), com pessoas familiarizadas com sistemas de gerenciamento de RH.

No contexto do Estudante podemos descrever que ele terá acesso de qualquer lugar e de qualquer dispositivo. Mas idealmente em um local privado, calmo e com dispositivo pessoal.
